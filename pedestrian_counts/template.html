D3 and maps

Reference materials are up here. Watch both videos along with reading and you'll be set.

Move the appropriate topojson/CSV files into the same folder as this file, then edit template.js to read in what you want. You don't HAVE to read both a csv and topojson in, but I thought it would be nice to give you a full template.

Server is probably run with python -m http.server, then you visit http://localhost:8000/.
